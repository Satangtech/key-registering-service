# key-registering-service

- Create file and correct data `app/.env` and `app/privkey.ts`

  - Copy file example

  ```bash
  cp app/.env.example app/.env
  cp app/privkey.example.ts app/privkey.ts
  ```

  - Config file `app/.env` and `app/privkey.ts`

- Start Service

  ```bash
  docker compose up -d
  ```
